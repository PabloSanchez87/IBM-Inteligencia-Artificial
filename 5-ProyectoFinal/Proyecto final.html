Una vez que tegas este dataset, puedes utilizar la biblioteca scikit-learn para separar los datos en grupos de entrenamiento y de evaluacin con la funcin train_test_split().

Primero, puedes crear un dataframe de pandas X con todas las columnas excepto la columna Class. Puedes crear una serie de pandas y solamente con la clumna Class column.

A continuacin, pueder usar la funcin train_test_split() para separar los datos en grupos de entrenamiento y de evaluacin. Puedes pasar las variables X e Y como argumentos junto con el argumento test_size para especificar el porcentaje del dataset que se deber asignar al grupo de evaluacin (en este caso, el 20%). Tambin puedes utilizar el argumento random_state para determinar un valor fijo de tal manera que, cada vez que se ejecute el cdigo, se produzca la misma divisin.

La funcin train_test_split() devuelve cuatro variables: X_train, X_test, y_train y y_test. X_train e y_train representan el grupo de entrenamiento, minetras que X_test e y_test representan el grupo de evaluacin. Puedes utilizar estas variable para entrenar y evaluar tus modelos de aprendizaje automtico.

Ten en cuenta que es importante dividir los datos en grupos de entrenamiento y de evaluacin para evitar el sobreajuste y permitir que se evale el rendimiento del modelo con datos que desconoce.
